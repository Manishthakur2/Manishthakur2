### 🔷 [CI/CD Pipeline — GitHub Actions + AWS EC2](https://github.com/Manishthakur2/CI-CD-pipeline)
> 4-stage automated pipeline that blocks broken code from ever reaching production.

- **Test** → pytest runs automatically, deployment blocked if anything fails
- **Build & Push** → Docker image built and pushed to DockerHub
- **Deploy** → SSH into EC2, replace running container automatically
- **Health Check** → hits /health endpoint to verify app is live
- Debugged real issues: container crash loops, SSH key permissions, Docker daemon access, AWS Security Group misconfigurations

**Stack:** GitHub Actions · Docker · AWS EC2 · Python Flask · pytest
